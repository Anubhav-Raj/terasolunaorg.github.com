TERASOLUNA Global Framework

What is TERASOLUNA Global Framework?

This framework is a comprehensive guide for WHAT, WHERE, WHEN, WHY and HOW for web development with (SpringMVC + Spring + JPA/MyBatis2) based technology stack. The primary purpose of this framework is to make it easier to develop spring based applications by providing the know-how which can only be achieved with years of experience in scalable java based web development.

Artifacts

TERASOLUNA Global Framework consists of following artifacts:

Development Guideline

Step-by Step Tutorials and intensive and focused best practices which takes in account numerous real world perculiar scenarios not to be found anywhere on the internet or reference books. See TERASOLUNA Global Framework Development Guideline.

Common Library

This is a library of useful and obstrusive common functionalities. Refer terasoluna-gfw to for further details about common library.

Template Blank Project

Blank template to quickly start with a solid foundation of project configuration and packing structure. Refer terasoluna-gfw-web-blank to get started with Template Blank Project.

Sample Application

Working example of all the best practices adviced in development guideline. Take inspiration directly from the code. Refer terasoluna-tourreservation for further details to get started with Reference Application.
